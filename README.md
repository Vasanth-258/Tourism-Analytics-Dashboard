# Tourism Analytics Dashboard

## Project Overview

This project analyzes tourism data using Power BI to provide insights into tourist arrivals, revenue, and seasonal trends.

## Tools Used

- Power BI
- Microsoft Excel
- Python
- Power Query
- DAX

## Dashboard Features

- KPI Cards
- State-wise Analysis
- Monthly Trend Analysis
- Revenue Analysis
- Interactive Filters
- Charts

## Dataset

Tourism dataset containing domestic tourists, foreign tourists, and tourism revenue.
